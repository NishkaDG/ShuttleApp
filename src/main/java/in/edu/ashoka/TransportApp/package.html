Contains the classes necessary for booking seats on Ashoka University Shuttles.

The classes contained are: class MainApplication (which runs the program), class everyDay and class Sweeper (which periodically update the programs), class Bookings (which enables booking, cancellation, etc), class Shuttle, which maintains the objects, class DBScribe (which maintains the databases) and so on.
